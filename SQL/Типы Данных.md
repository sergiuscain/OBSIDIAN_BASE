##  Типы Данных

| Тип         | Пример                  | Описание                        |
| ----------- | ----------------------- | ------------------------------- |
| INT         | `1234`                  | Целое число                     |
| VARCHAR(50) | `'Текст'`               | Строка переменной длины         |
| DECIMAL     | `999.99`                | Число с фиксированной точностью |
| DATETIME    | `'2023-06-21 14:30:00'` | Дата и время                    |
| JSON        | `{"key": "value"}`      | JSON-данные                     |
|             |                         |                                 |
