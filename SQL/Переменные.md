Перед названием переменных ставим символ '@'
Для присвоения значения переменной пишем:
SET имя_переменной = значение;
```sql
SET @count = 10; SELECT @count;
```
Использование переменных:
```sql
SET @threshold1 = 90000, @threshold2 = 100000; SELECT * FROM employees WHERE salary BETWEEN @threshold1 AND @threshold2;
```
