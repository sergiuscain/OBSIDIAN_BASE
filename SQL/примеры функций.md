# GROUP_CONCAT

^groupconcat

У функции есть дополнительные параметры в следующем порядке:

- DISTINCT: Указывает, что дубликаты должны быть удалены из результирующего списка.
- ORDER BY: Определяет порядок сортировки значений перед их объединением.
- SEPARATOR: Указывает строку, которая будет использоваться в качестве разделителя между значениями. По умолчанию используется запятая.
```sql
SELECT GROUP_CONCAT(DISTINCT name ORDER BY name SEPARATOR '; ') FROM employees;
```
```sql
+----------------------------------------------------------------+ 
| GROUP_CONCAT(DISTINCT name ORDER BY name SEPARATOR '; ')       | 
+----------------------------------------------------------------+ 
| Jack; James; Jane; Janet; Jessica; Jill; Jim; Joe; John; Julie | 
+----------------------------------------------------------------+
```
