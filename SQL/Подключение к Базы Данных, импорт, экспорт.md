## Подключение к Базы Данных, импорт, экспорт

### -- Подключение к серверу
```sql
mysql -u [user] -p[password] -h [host] -P [port]
```

### -- Создание БД
```sql
CREATE DATABASE db_name;
```

### -- Переключение на БД
```sql
USE db_name;
```

 ### -- Экспорт всей БД
```sql
mysqldump -u user -p db_name > backup.sql
```

 ### -- Импорт БД
```sql
mysql -u user -p db_name < backup.sql
```
