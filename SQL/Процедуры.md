Синтаксис процедуры
```sql
DELIMITER // 
CREATE PROCEDURE GetAllEmployees(IN emp_id INT) 
BEGIN SELECT * FROM employees; 
END // 
DELIMITER ;
```

Параметры процедуры могут быть трех типов:

- IN: Параметры, которые передаются в процедуру;
- OUT: Параметры, которые возвращаются из процедуры;
- INOUT: Параметры, которые могут быть использованы как входные, так и выходные.
