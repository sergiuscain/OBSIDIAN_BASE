## -Virta
Virtal метод - это метод, который имеет реализацию базового класса, но может быть переопределен наследником.
В базовом классе пишем:
public virtual void MakeSound() // реализация по умолчанию
{
	Console.WriteLine("Make sound...");
}
В наследники мы может переопределить метод MakeSound:
public override void MakeSound()
{
	Console.WriteLine("Гав-гав"); 
	// Собака наследовалась от класса животного и теперь она может произносить звук, как и все животные. Но собака переопределила своё произношение звука. Теперь она гавкает. 
}

Виртуальный метод может быть в любом классе.

## - abstract
Abstract метод - это метод без реализации. Но каждый наследник ОБЯЗАН реализовать этот метод сам.
public abstact double GetArea(); // Нет реализации. Только сигнатура.

В наследнике:
public double Radius {get; set;}
public override double GetArea()
{
	Return Math.PI * Radius * Radius;
}

Абстрактный метод может быть только в абстрактном классе!
