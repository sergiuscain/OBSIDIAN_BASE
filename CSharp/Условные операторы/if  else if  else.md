# Описание
`if`, `else if`, `else` - это конструкция для ветвления программы в зависимости от условия.

# Синтаксис
Блок if принимает булевое значение (условие, проверка чего-то. Что угодно, что может быть истиной или ложью).
Например (возраст > 20). Если возраст больше 20, то это выражение вернёт true и код в блоке if выполнится. Далее идут необязательные блоки.
else if (Иначе если) выполнится в том случае, если блок выше (if или другой else if) не выполнился. Блоков else if может быть сколько угодно.
Далее выполняется необязательный блок else. Код в этом блоке выполнится только, если все условия выше не прошли проверку (везде false).
```csharp
int age = 20;

if (age < 18)
{
    Console.WriteLine("Вы несовершеннолетний");
    Console.WriteLine("Здесь может быть другой код");
}
else if (age >= 18 && age < 65)
{
    Console.WriteLine("Вы взрослый");  // Выполнится этот блок
}
else
{
    Console.WriteLine("Вы пенсионер");
}
```
## Краткая форма для одной команды
В данном коде мы опускаем фигурные скобки так как для каждого условия только одна инструкция.
```csharp
int x = 10;
int y = 5;

// Для одной инструкции скобки можно опустить
if (x > y)
    Console.WriteLine("x больше y");  // Выполнится
else
    Console.WriteLine("x не больше y");
```
Мы можем сделать и так, но это будет смотреться некрасиво:
```csharp
bool conditionName = x > y;
bool condition2 = age >= 18 && age < 65;
bool conditionName = x > y;
if (conditionName)
    Console.WriteLine("x больше y"); 
else if (condition2)
{
    Console.WriteLine("Вы взрослый");  
}
else if (age >= 18 && age < 65)
    Console.WriteLine("Вы взрослый");  
else if (age >= 18 && age < 65)
{
    Console.WriteLine("Вы взрослый");  
}
else
    Console.WriteLine("x не больше y");
```
То есть, фигурные скобки, в случае, если инструкция ровно одна, опциональны. 

# Может быть полезно
## Тернарный оператор
Если у вас есть простое условие, вы можете использовать такую конструкцию:
condition ? value1 : value2
```csharp
        int a = int.Parse(Console.ReadLine());
        int b = int.Parse(Console.ReadLine());
        Console.WriteLine(a > b ? "a больше b" : "b больше a");
```
Вначале пишем условие (a > b), потом знак '?', потом то, что вернётся, если условие верное, потом знак ':', потом то, что вернётся, если условие ложно.
Такую конструкцию можно разложить на следующий код:
```csharp
if (a > b)
	return "a больше b";
else
	return "b больше a";
```
