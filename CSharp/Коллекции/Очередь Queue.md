# Описание
- Queue— это структура данных, которая работает по принципу "первым пришёл — первым ушёл" (FIFO). Аналогия: живая очередь в больницы. Кто первый пришел, тот первым и попадёт на приём.
- Очередь удобно перебирать через Foreach
- Если с помощью методов Peek или Dequeue попытаться получить первый элемент очереди, которая пуста, то программа выдаст исключение. Перед получением элемента можно проверять количество элементов в очереди.
# Создание очереди
```csharp
using System.Collections.Generic;

Queue<int> numbers = new Queue<int>(); 
Queue<string> strings = new Queue<string>();
```
# Основные методы
Enqueue - Добавляет элемент в конец очереди.
Dequeue -Извлекает и возвращает первый элемент очереди.
Clear - Очищает очередь.
Contains - Проверяет наличие элемента в очереди.
Peek - Возвращает первый элемент из начала очереди без его удаления.
