# Описание
Стэк - это структура данных типа  **LIFO** (last in, first out — последним пришел, первым ушел).
Аналогия: стопка посуды. Последнюю тарелку положили, последнюю и берём (из середины стопки нельзя достать). То есть, в стэке у нас есть доступ только к верхнему (последнему )элементу!!

# Создание Стэка
```csharp
using System.Collections.Generic;
....
Stack<int> numbers = new Stack<int>(); 
Stack<string> strings = new Stack<string>();
...
```
# Методы Стэка
Push - **Добавляет элемент в вершину стека**.
Peek - **Возвращает элемент из вершины стека, не удаляя его при этом**.
Pop - **Удаляет и возвращает элемент из вершины стека**.
