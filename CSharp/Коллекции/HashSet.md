# Описание
- Класс в C#, который предназначен для хранения коллекции уникальных элементов. Обозначается как 
- **Не поддерживает порядок элементов**. Focus на уникальности элементов и быстром поиске. 
- **Повторяющиеся элементы не могут быть добавлены**. Если они добавлены, внутренне игнорируются без выдачи ошибки.
- **Реализован через хэш-таблицу**.
- **Обеспечивает быстрый поиск, добавление и удаление** (в среднем O(1) времени).
- **Реализует интерфейс ISet<T**>, что позволяет выполнять операции на наборах, например, объединение, пересечение. 
# Создание HashSet
```csharp
using System.Collections.Generic;

HashSet<int> numbers = new HashSet<int>(); 
HashSet<string> strings = new HashSet<string>();
```

# Методы HashSet
- **Add** — добавляет указанный элемент в HashSet, если он не присутствует. Если элемент уже присутствует, метод ничего не делает и возвращает false.
- **Clear** — удаляет все элементы из HashSet.
- **Contains** — определяет, содержит ли HashSet указанный элемент. Возвращает true, если элемент найден в наборе, иначе — false.
- **UnionWith** — объединяет два HashSet, создавая новый HashSet, содержащий все уникальные элементы из обоих наборов.
- **IntersectWith** — находит общие элементы между двумя HashSet и обновляет первый HashSet этими элементами.
- **ExceptWith** — вычитает элементы одного HashSet из другого, изменяя первый HashSet результатом.
