# Описание

- Класс List представляет из себя динамически изменяемый массив. 
- **Поддержка индексов** — с их помощью можно обратиться к определённым элементам.
- **Указание начальной ёмкости** — конструктор класса List принимает в качестве параметра начальную ёмкость списка, что позволяет увеличить производительность и уменьшить издержки на выделение памяти при добавлении элементов. 
- **Обращение к элементам** — с помощью свойства Count можно получить длину списка.
- **Перебор списка** — можно использовать классический циклы, например for, или foreach.
- 

# Создание листа
```csharp
List<T> = new List<T>();
List<string> people = new List<string>();
```

# Некоторые Методы
```csharp
- **Add(T item)** — добавляет новый элемент с типом T в конец списка.
- **AddRange(IEnumerable<T> collection)** — добавляет в список коллекцию или массив в конец списка.
- **Insert(int index, T item)** — вставляет элемент в коллекцию по указанному индексу. При выходе за границы коллекции (0 > index или Count-1 < index) будет сформировано исключение System.ArgumentOutOfRangeException.
- **InsertRange(int index, IEnumerable<T> collection)** — аналогично методу выше, только вставляет список элементов по указанному индексу.
- **BinarySearch(T item)** — бинарный поиск элемента в списке, возвращает индекс элемента. Работает корректно, если список отсортирован.
- **Remove(T item)** — удаляет элемент item из списка, и если удаление прошло успешно, возвращает true. Если в списке несколько одинаковых элементов, то удаляется только первый из них.
  **RemoveAt(int index)** — удаление элемента по индексу.
```
