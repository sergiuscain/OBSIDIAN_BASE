# Описание
Цикл `for` используется, когда известно точное количество итераций. Он состоит из трех частей: инициализации, условия и итератора.
for (инициализация; условие; итератор)
{
    // тело цикла
}
# Особенности for

- **Полный контроль** над индексом и шагом
- **Можно изменять коллекцию** во время итерации
- **Эффективен** для массивов по индексу
- **Гибкая настройка** всех параметров цикла
# Синтаксис
Мы создаем переменную int i; Далее пишем условие, при котором выполняется тело цикла;
Итератор (делаем какие-то действия с нашей переменной). i++ / i-- / i/2*10 и так далее.
```csharp
for (int i = 0; i < 5; i++)
{
    Console.WriteLine($"Итерация: {i}");
}
// Вывод: 0, 1, 2, 3, 4
```
Мы можем пройтись по массиву с шагов 2. 
```csharp
// Вывод четных чисел
for (int i = 0; i <= 10; i += 2)
{
    Console.WriteLine(i);  // 0, 2, 4, 6, 8, 10
}
```
