`docker run rabbitmq` - запускает контейнер по названию образа и скачивание образа в случае его отсутствия. rabbitmq - образ, который находится на Docker Hub.
`docker pull rabbitmq` - скачивает образ, но не запускает его.
`docker ps -a` - показывает список контейнеров. В том числе и не запущенных. 
`docker ps` - показывает запущенных список контейнеров.
`docker start 36bfbd866a06` - запуск контейнера. "36bfbd866a06" - ID контейнера, который можно посмотреть командой `docker ps -a`
`docker stop 36bfbd866a06` - останавливает контейнер с выбранным ID.
`docker rm 36bfbd866a06` - удаляет контейнер.
`docker rmi [название образа]` - удаляет образ (при этом контейнеры этого образа не должны быть запущенны)

`docker inspect [id контейнера]` - выводит информацию о контейнере в JSON формате
