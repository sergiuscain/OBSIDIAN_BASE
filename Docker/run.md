`docker run redis:5.0` - после названия образа пишем двоеточие и версию образа.
После `:` мы пишем теги. docker ищет образ с нужным тегом и скачивает его и создает контейнер по этому образу.
У последней версии образа как правило есть тег `latest`, который указывает, что это последняя версия образа. Если мы не указываем никакой тег, по умолчанию скачивается последняя версия образа (latest)
